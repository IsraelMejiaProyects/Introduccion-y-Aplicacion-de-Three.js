Fundamentos de Three.js

Aprende los conceptos esenciales para crear escenas 3D en la web.

¿Qué es Three.js?

Three.js es una librería de JavaScript que permite crear gráficos 3D en el navegador utilizando WebGL. Facilita la creación de escenas, cámaras, luces y objetos sin necesidad de trabajar directamente con WebGL.

Escena (Scene)

La escena es el contenedor principal donde se agregan todos los objetos 3D, luces y cámaras. 

const scene = new THREE.Scene();

Cámara (Camera)

La cámara define el punto de vista desde el cual se observa la escena. 

const camera = new THREE.PerspectiveCamera(75, width/height, 0.1, 1000);

Renderizador (Renderer)

El renderer se encarga de dibujar la escena en el navegador. 

const renderer = new THREE.WebGLRenderer();

Geometría

Define la forma del objeto 3D (cubo, esfera, etc.). 

const geometry = new THREE.BoxGeometry();

Material

Define la apariencia del objeto, como color, textura y brillo. 

const material = new THREE.MeshStandardMaterial({ color: 0x00ffcc });

Luz

La iluminación permite ver los objetos con profundidad y realismo. 

const light = new THREE.PointLight(0xffffff, 1);

Objeto (Mesh)

Un objeto 3D se crea combinando geometría y material. 

const mesh = new THREE.Mesh(geometry, material);
